## 前端文件夹

- `src`：存放前端源代码，如组件、页面、样式表、脚本等
- `public`：存放静态资源
- `package.json`：前端项目的依赖和脚本
- `webpack.config.js`：Webpack 配置文件

## 服务端文件夹

- `src`：存放服务端源代码，如路由、控制器、模型、服务等
- `config`：存放配置文件
- `package.json`：服务端项目的依赖和脚本
- `server.js`：服务端入口文件
